Ruby Basics

    Integers

  1. 1.

    An integer is a number 1 , 2 , 3 ,4 , 5

    If you want to see if an integer is even, you can used some fancy methods:

    4.even? will return true and 4.odd? will return false

    If you have a string like "123" and want to make it an integer, you can use .to_i

    2. Arithmetic Operators

  2. 2.

    Let's look at math operators assuming a=5 and b=2

    To add two variables together, use the + sign (a+b) => 7

    To add two variables together, use the - sign (a-b) => 3

    To multiply variables together, use the * sign (a*b) => 15

    To divide variables , use the / sign (a/b) => 2

    *The above example returns 2 and not 2.5 because integers return full numbers and indicate how many complete times one number goes into another*

    To find a modulus, use %. This will give us the remainder.

    3. Comparison Operators

  3. 3.

    Comparison operators return true or false. Assume a=5 and b=2

    == checks to see if two integers are equal. a==b => false

    != checks to see if two integers are NOT equal. a!=b => true

    > and < check if something is greater than or less than. a>b => true

    >= and <= check if something is greater or equal to or than or less than. a>=b => true

    4. Strings

  4. 4.

    A string is anything in " "

    If you want to manipulate a string, you can used some fancy methods:

    "string".capitalize => "STRING"

    "string".length => 6

    "string".reverse => "gnirts"

    To create an empty string, string = " "
